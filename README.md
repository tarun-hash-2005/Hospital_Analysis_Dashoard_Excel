Hospital_Analysis_Dashboard_Excel

🔍 Project Overview
This project presents an interactive **Hospital Analysis Dashboard** built using Microsoft Excel. The dashboard provides insights into hospital operations, patient trends, and revenue performance to support data-driven decision-making in the healthcare domain.

---

🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Data Cleaning & Transformation
- Slicers for interactivity
- Data Visualization & Dashboard Design

---

 📊 Key Features of Dashboard
- 💰 Total Revenue analysis
- 🩺 Count of Diagnosis overview
- 🏥 Department-wise patient admissions
- 📈 Patients admitted & discharged trend (monthly)
- 👨‍⚕️ Doctor-wise patient handling analysis
- 💳 Payment status (Paid vs Pending)
- 👩‍⚕️ Gender-wise patient distribution
- 💸 Revenue contribution by department
- 🎛 Interactive filters (Department & Gender slicers)

---

 
 📈 Business Insights
- Identified top revenue-generating departments
- Analyzed patient admission and discharge trends
- Evaluated doctor workload distribution
- Observed payment patterns (Paid vs Pending)
- Compared gender-wise patient distribution

---

 📂 Dataset Details
The dataset includes:
- Patient Information
- Department Details
- Doctor Names
- Admission & Discharge Data
- Revenue & Payment Status

---


 💡 Skills Demonstrated
- Data Cleaning & Preparation
- Data Analysis using Excel
- Dashboard Design & Visualization
- Business Insight Generation

##  🚀 Conclusion
This dashboard helps healthcare management understand operational performance, optimize resource allocation, and improve decision-making using data-driven insights.
